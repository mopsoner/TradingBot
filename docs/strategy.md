# Strategy Specification

## Long setup
1. Visible liquidity below price
2. Sweep below support / equal lows / range low
3. Spring classification
4. Bullish displacement
5. Bullish BOS
6. Retracement into Fibonacci 0.5 / 0.618 / 0.705
7. Expansion target = next visible liquidity

## Short setup
1. Visible liquidity above price
2. Sweep above resistance / equal highs / range high
3. UTAD classification
4. Bearish displacement
5. Bearish BOS
6. Retracement into Fibonacci 0.5 / 0.618 / 0.705
7. Expansion target = next visible liquidity

## No-trade conditions
- no sweep
- no displacement
- no BOS
- no fib entry
- 1H / 4H conflict
- invalid stop placement
